Tersesat di Ilmu Komputer

Published on April 22, 2025 • By Lekok Indah Lia

Saya masuk jurusan Ilmu Komputer karena berpikir kuliahnya akan seperti belajar Microsoft Word, Excel, atau PowerPoint. Saya pikir, "Oh, ini pasti seperti belajar komputer di sekolah dulu." Ternyata… saya salah besar.

Di semester pertama, saya langsung dihadapkan dengan istilah-istilah yang asing seperti algoritma, flowchart, struktur data, dan tentu saja: coding. Waktu pertama kali buka Visual Studio Code dan melihat layar hitam penuh tulisan putih, saya jujur merasa sangat minder. Banyak teman-teman yang sudah familiar dengan dunia pemrograman, sementara saya bahkan belum tahu cara print "Hello World".

Saya sempat merasa tidak cocok, bahkan berpikir apakah saya salah jurusan. Tapi seiring waktu, saya mulai menemukan sisi lain dari dunia Ilmu Komputer, sisi yang lebih visual, lebih manusiawi, dan lebih kreatif: UI/UX Design.

Dari situ saya mulai belajar secara mandiri. Saya mengikuti kursus online, nonton video, dan mencoba membuat desain tampilan sederhana. Ternyata, saya menyukainya! Saya mulai menyadari bahwa menjadi bagian dari dunia teknologi tidak harus jago ngoding. Ada juga peran penting seperti desainer antarmuka dan pengalaman pengguna dan itu adalah bidang yang membuat saya semangat lagi.

Mungkin saya memang sempat "tersesat", tapi dari kesesatan itulah saya justru menemukan arah baru. UI/UX membuat saya percaya bahwa saya tetap punya tempat di dunia Ilmu Komputer, meskipun bukan di jalur yang awalnya saya bayangkan.

Dan sekarang, saya tidak lagi takut salah jurusan ~ saya hanya sedang mencari jalur saya sendiri.
